## 📱 Cara Mengubah Menjadi Aplikasi HP (APK)
Anda dapat mengubah kode web di repositori ini menjadi aplikasi Android matang tanpa perlu keahlian *coding* lanjutan:
1. Aktifkan **GitHub Pages** melalui menu *Settings* > *Pages* di repositori ini.
2. Salin tautan domain web yang disediakan oleh GitHub Pages.
3. Buka situs konverter gratis seperti **[Webintoapp](https://webintoapp.com)** atau **[Median.co](https://median.co)**.
4. Masukkan tautan GitHub Pages Anda, tentukan nama aplikasi serta ikon, lalu unduh file `.apk` yang dihasilkan.
